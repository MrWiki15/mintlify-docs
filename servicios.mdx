Kawi ofrece servicios enfocados en movimiento de dinero, conversión y liquidación.

## Servicios principales

Los servicios principales son:

- Depósitos en moneda local.
- Retiros en moneda local.
- Conversión fiat → cripto.
- Conversión cripto → fiat.
- Conversión cripto → cripto.
- Transferencias fiat → fiat.
- Remesas.
- Pagos.
- Compras.
- Infraestructura API para empresas.
- Widgets embebibles.

## Cómo depositar en Kawi

Un depósito es la entrada de fondos a la plataforma.

### Flujo de ejemplo

```text
Usuario deposita BRL vía PIX
↓
Kawi detecta el pago
↓
El saldo se acredita en la plataforma
↓
El usuario puede convertir, enviar o utilizar sus fondos
```

### Qué debe hacer el usuario

1. Ir a la sección **Depositar**.
2. Seleccionar el método disponible.
3. Seguir las instrucciones de pago.
4. Confirmar la operación.
5. Esperar la acreditación automática.

### Qué ocurre internamente

Kawi utiliza sistemas de pago fiat para recibir el dinero del usuario. Luego puede conectar ese valor con su motor de liquidación para ejecutar conversiones o transferencias.

## Cómo retirar en Kawi

Un retiro es la salida de fondos desde Kawi hacia un destino externo.

Los retiros pueden enviarse a:

- Una cuenta bancaria.
- Una wallet cripto.
- Otro usuario.
- Un beneficiario final.
- Un proveedor conectado.

### Flujo de ejemplo

```text
Usuario solicita retiro
↓
Kawi valida la operación
↓
El motor selecciona la ruta adecuada
↓
Se ejecuta la salida
↓
El usuario recibe confirmación
```

## Cómo gestionar fondos

Dentro de Kawi, el usuario puede gestionar sus fondos de forma simple.

Esto incluye:

- Consultar saldo.
- Ver historial de operaciones.
- Convertir saldo.
- Enviar fondos.
- Recibir fondos.
- Consultar estado de transacciones.
- Descargar comprobantes.

Kawi busca que la gestión de fondos sea intuitiva, incluso para usuarios sin experiencia financiera o cripto.

## Compras dentro de Kawi

Kawi también puede permitir compras de productos o servicios dentro del ecosistema.

Ejemplos:

- Compra de servicios digitales.
- Compra de productos para familiares.
- Pagos a comercios.
- Servicios integrados de terceros.
- Productos ofrecidos por aliados.

La sección de compras permite que Kawi evolucione más allá de remesas. Así se convierte en una plataforma de servicios financieros y comerciales conectada a una infraestructura de liquidación.
