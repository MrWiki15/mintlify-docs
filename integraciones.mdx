Kawi puede integrarse con empresas que necesitan mover dinero entre sistemas, monedas o países.

## Remesas como Servicio

Remesas como Servicio permite que una empresa integre la infraestructura de Kawi sin construir desde cero su propia operación de remesas.

### Qué ofrece

- API para crear operaciones.
- Widget embebible.
- Conversión fiat ↔ cripto.
- Liquidación transfronteriza.
- Notificaciones de estado.
- Gestión de beneficiarios.

### Flujo de ejemplo

```text
Empresa integra Kawi
↓
Usuario de la empresa inicia operación
↓
Kawi procesa depósito, conversión y liquidación
↓
La empresa muestra el estado al usuario
```

## Integración con empresas

Kawi puede integrarse con empresas que necesitan mover dinero entre sistemas.

Ejemplos de empresas:

- Fintechs.
- Startups.
- Marketplaces.
- Plataformas de servicios.
- Apps de comunidades migrantes.
- Empresas con usuarios internacionales.

## Casos de uso B2B

### Fintech

Una fintech puede usar Kawi para ofrecer conversión fiat ↔ cripto sin construir infraestructura propia.

### Marketplace

Un marketplace puede permitir pagos internacionales y liquidación a vendedores.

### App de remesas

Una app de remesas puede usar Kawi como backend de liquidación.

### Plataforma comunitaria

Una comunidad puede integrar Kawi para permitir pagos, envíos y servicios financieros a sus usuarios.
