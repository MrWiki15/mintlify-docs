Kawi permite que empresas y desarrolladores integren capacidades de pagos, conversión y liquidación directamente dentro de sus productos.

Con la API de Kawi, una empresa podra:

- Crear depósitos.
- Consultar balances.
- Ejecutar conversiones.
- Enviar fondos.
- Integrar un widget embebible.
- Usar Kawi como infraestructura de remesas o liquidación.

## Autenticación

Todas las peticiones a la API de Kawi requieren autenticación mediante API key.

Para más información -> [Portal de desarrolladores](https://kawiservices.com.br/api/docs).
