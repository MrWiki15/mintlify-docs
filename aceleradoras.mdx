Kawi participa en programas de aceleración para fortalecer su infraestructura, su producto y su relación con el ecosistema Web3.

## WayLearn

Kawi fue seleccionado para participar en WayLearn x Solana Foundation.

### Qué buscamos conseguir

En esta aceleradora, Kawi busca:

- Refinar la arquitectura de liquidación sobre Solana.
- Fortalecer el uso de stablecoins dentro del producto.
- Validar el motor de liquidación híbrida.
- Mejorar la narrativa técnica de Kawi como infraestructura.
- Conectar con mentores, builders y oportunidades dentro del ecosistema Solana.

### Objetivos dentro de WayLearn

1. Refinar el motor de liquidación.
2. Procesar más de 10k USD usando el flujo mejorado.
3. Alcanzar 10k transacciones en el motor.
4. Preparar una integración técnica más sólida sobre Solana.
5. Consolidar Kawi como infraestructura real dentro del ecosistema.

| --- | --- |
| 1 | Terminar la infraestructura completa de settlement. |
| 2 | Ejecutar las primeras 50 transacciones en el motor. |
| 3 | Procesar los primeros 100 USD en el motor. |
| 4 | Alcanzar las primeras 100 transacciones en el motor. |
| 5 | Procesar los primeros 1k USD. |
| 6 | Alcanzar 3k transacciones. |
| 7 | Procesar 10k USD. |

## Dev3Pack

Kawi también participa en Dev3Pack como parte de su proceso de aceleración, validación y conexión con el ecosistema Web3.

### Qué buscamos conseguir

En Dev3Pack, Kawi busca:

- Fortalecer la estructura de producto.
- Mejorar el posicionamiento como infraestructura financiera.
- Preparar el proyecto para oportunidades de inversión.
- Refinar el pitch y la documentación.
- Aumentar la claridad del roadmap técnico y comercial.
- Conectar con partners estratégicos.

### Objetivos dentro de Dev3Pack

1. Convertir el producto en una infraestructura más clara para terceros.
2. Preparar la documentación de API y widget.
3. Validar el modelo B2B de Remesas como Servicio.
4. Medir adopción, recurrencia y volumen.
5. Preparar una presentación sólida para futuras aceleradoras o fondos.

### Roadmap dentro de Dev3Pack

| Fase | Meta |
| --- | --- |
| 1 | Diagnóstico del producto y narrativa. |
| 2 | Documentación técnica y comercial. |
| 3 | Preparación de demo y pitch. |
| 4 | Validación de partners. |
| 5 | Definición de métricas y objetivos de crecimiento. |
| 6 | Preparación para escalamiento. |
