---
title: "Cómo funciona Kawi"
description: "Entiende cómo Kawi conecta fiat, blockchain y stablecoins en flujos simples para usuarios y empresas."
---

Kawi funciona conectando dos mundos:

1. **Fiat:** dinero tradicional, como reales, dólares u otras monedas locales.
2. **Blockchain:** redes y activos digitales como Solana, USDC o USDT.

La idea central es:

```text
Fiat para entrar
Blockchain para liquidar
```

Kawi actúa como una capa de orquestación. Coordina pagos, conversiones, liquidez, rutas y confirmaciones para que el usuario no tenga que hacerlo manualmente.

## Fiat → cripto

Este flujo permite convertir moneda local en activos digitales.

### Ejemplo

```text
BRL vía PIX
↓
Kawi recibe el pago
↓
Kawi ejecuta conversión
↓
Usuario recibe USDC o saldo equivalente
```

### Para qué sirve

- Entrar al ecosistema cripto.
- Enviar valor globalmente.
- Usar stablecoins.
- Liquidar operaciones internacionales.

### Beneficio

El usuario no necesita usar un exchange complejo ni aprender sobre órdenes de mercado, redes o wallets.

## Cripto → fiat

Este flujo permite convertir activos digitales en moneda local.

### Ejemplo

```text
Usuario envía USDC
↓
Kawi recibe el activo digital
↓
Kawi ejecuta conversión
↓
Usuario recibe fiat
```

### Para qué sirve

- Retirar stablecoins a moneda local.
- Pagar a usuarios no cripto.
- Convertir ingresos digitales en dinero usable.
- Completar remesas o liquidaciones.

## Cripto → cripto

Este flujo permite convertir un activo digital en otro.

### Ejemplo

```text
USDC
↓
Swap interno
↓
USDT
```

### Para qué sirve

- Optimizar rutas de liquidez.
- Usar distintos activos según el destino.
- Mejorar eficiencia de liquidación.
- Conectar diferentes redes o tokens.

Actualmente, este flujo se apoya en agregadores como Jupiter para encontrar mejores rutas dentro de Solana.

## Fiat → Fiat (Core)

Este flujo permite mover valor entre monedas tradicionales usando Kawi como infraestructura intermedia.

### Ejemplo

```text
Usuario deposita moneda local
↓
Kawi procesa internamente la operación
↓
Beneficiario recibe moneda local
```

### Beneficio

El usuario ve una experiencia simple de envío o pago, mientras Kawi maneja internamente la conversión, liquidez y liquidación.
